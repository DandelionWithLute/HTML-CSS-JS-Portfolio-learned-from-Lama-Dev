两个也是相对单位不过其相对的是视口（非body标签定义的宽度）的，

vw： viewport width 意思是视口的宽度
vh： viewport height 意思是视口的高度。
现在说一下具体的单位大小都是各自的百分之一。

1vw = 1/100 * 视口宽度。
1vh =1/100 * 视口高度
